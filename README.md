# change File Format

Scripts to easily change text and image formats.

## How to Use

To use these scripts, make sure you have python installed and then follow these steps:

1. **Clone the Repository**:

    First, clone the repository to your local machine by running:

    ```bash
    git clone https://github.com/s1lvaj/change-file-format.git
    ```

2. **Download the Repository (Optional)**:

    Alternatively, you can download the repository directly without using Git. Simply append `/zipball/master` to the end of the URL:

    ```bash
    https://github.com/s1lvaj/change-file-format/zipball/master
    ```

3. **Run the Script**:

    Navigate to the project directory and run any of the scripts you want, and it'll affect your files.
